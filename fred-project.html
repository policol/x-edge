Edge Timer

How to play

Overview

Edge Timer guides you through a session of repeated edging cycles. Each cycle has three phases: build-up, edging, and rest. Complete all edges to finish the session.

Session Phases

  • Pre-Edge — A countdown runs. Build up stimulation and press Edge when you reach the edge. Pressing early is fine; a late press adds bonus time to your next edge.
  • Edging — Hold at the edge for the full duration. The button is locked — you must stay until the timer ends.
  • Resting — Cool down for the displayed duration. The next cycle starts automatically when rest ends.

Edge Failed

If you go over the edge during the Edging phase, press ⚠ Edge Failed at the bottom of the screen. Penalties are applied based on how much time was left on the timer:

Time Remaining Penalties Applied
< 30s +5% rest skip chance
30s – 1 min +5% skip chance, edge max +1 min
1 – 2 min +10% skip chance, edge +1/+2 min, rest ×0.90
2 – 3 min +20% skip chance, edge +1/+2 min, rest ×0.80
3 – 4 min +20% skip chance, edge +2/+2 min, rest ×0.70
4+ min +40% skip chance, edge +3/+3 min, rest ×0.10, +1 extra edge

Status Panel

Once a session starts, a panel below the progress bar shows all live parameters — edge ranges, rest ranges, skip chance, and the next rest modifier. Penalty rows dim when inactive and light up in orange when a penalty is pending.

Config

Tap Config (top-left) before starting to adjust all ranges manually, or use Randomize Parameters to generate a random session. Config is locked while a session is running.

Controls

  • Pause / Resume — freezes the current timer.
  • Restart — ends the session and resets all penalties.
